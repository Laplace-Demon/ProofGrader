要证：
对于任意 $\{ a_n, n \in N \}$，若 $\{ a_n, n \in N \}$ 收敛，则 $\{ a_n, n \in N \}$ 有界

证明：

假设 $\{ a_n, n \in N \}$ 收敛,

根据收敛的定义，$存在l$ 满足 $\lim\limits_{n \to +\infty}{a_n} = l$,

根据极限的定义，$存在N$ 满足 对于任意$n$，若$n \ge N$，则$|a_n - l| < 1$,

因此，对于任意$n$，若$n \ge N$，则$|a_n| = |a_n - l + l| < |a_n - l| + |l| < 1 + l$,

根据有界定义可得，数列 $\{ a_n, n \in N \}$ 有界,

综上可知，原命题得证。
